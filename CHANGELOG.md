# Changelog

### Added
- Initial release
- IEEE 802.15.4 MAC layer parsing
- Zigbee NWK/APS/ZCL support
- TI CC2531/CC2652 drivers
- ConBee driver
- Network topology mapping
- Traffic statistics
- Security analysis
- Anomaly detection
- Channel analysis
- Device fingerprinting
- Terminal UI with 7 tabs
- CLI with 4 commands
- Auto-device detection
- Real-time monitoring

### Technical
- ~15,000 lines of Rust
- Async packet capture
- <10ms latency
- Up to 250 pkt/sec
- ~50MB memory usage
